# 🎓 UniConnect - Réseau Social Académique Éthique

**UniConnect** est un MVP développé dans le cadre d'un projet de 3ème année. C'est un réseau social "Multi-Tenant" conçu pour favoriser l'entraide entre étudiants tout en respectant des principes d'ingénierie éthique.

## 🌟 Fonctionnalités Clés
- **Sécurité Multi-Domaine :** Accès réservé aux emails académiques.
- **Isolation des Données :** Chaque université possède son propre espace privé.
- **Mode Focus :** Interface limitée entre 22h et 07h pour le bien-être des étudiants.
- **Bibliothèque Partagée :** Dépôt de ressources (PDF, liens) par thématique.

## 🛠️ Stack Technique
- **Framework:** Laravel 11 & React (Inertia.js)
- **Style:** Tailwind CSS
- **Base de données:** SQLite (Portable)

## 🚀 Installation rapide
1. `composer install`
2. `npm install && npm run build`
3. `php artisan migrate`
4. `php artisan serve`
